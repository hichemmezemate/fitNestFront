FITNEST

Avec le créateur d'entraînements, votre coach peut facilement créer le plan d'entraînement que vous convient. Le plan d'entraînement crée pour vous vous montre comment les exercices sont effectués et quels groupes musculaires ils sollicitent. Le plan d'entraînement personnalisé que vous avez conçu avec le générateur est automatiquement sauvegardé dans votre profil. De cette façon, lorsque vous allez vous entraîner, vous pouvez ouvrir votre page de profil et suivre le plan d'entraînement que vous avez enregistré.
